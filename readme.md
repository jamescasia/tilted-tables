# Tilted Tables

Tilted Tables is a fun and cute 3d puzzle game wherein one rotates a "tilted table" to let Cubie escape! This game uses [Web Monetization](https://webmonetization.org/) through [Coil](https://Coil.com). This game was built in 6 days for the Web Monetization Game Jam.
